THREE REGISTERED VERSIONS

There are several different registered packages available at different price levels, depending on what you need or want. There are three different Versions of the Program.All registered versions include the 1200 location Extended Location Database. This contains mainly U.S. and Canadian Locations, but has some other locations as well.

LITE VERSION

A LITE Version which is basically this version. This is mainly provided for Beginners, and those who don't need the advantages of the more advanced versions. If you send $44.95 you will get this version. This registration level only provides the disks, No printed manual is provided. Registering this Package entitles you to access to technical support for 90 days.

DELUXE VERSION

A DELUXE Version Which offers all the features that The Lite Version does, but adds more Reports and adds other Features and supports More printers. This version costs $59.95 and Includes a Printed Manual. Most users will find this version best for their needs. Registering this Package entitles you to access to technical support for 90 days.

FEATURES OF DELUXE VERSION IN ADDITION TO LITE FEATURES.

* Progressions Report.
* Solar and Lunar Returns.
* Composite Report.
* Relocations Report.
* Midpoints.
* Click on Glyphs and see interpretive text.
* Display of Planet Locations.
* Customize Graphs with your name and address.

In addition to these features, for an additional $25, I will send you the Text Source files used to create ASTRO's Text interpretations. These are in ASCII. You can edit these files with any Word Processing program that accepts plain text or ASCII. This builds these files into ASTRO's Proprietary Indexed Compressed format. As a special Offer, You can order both the DELUXE Program and the Text Source and Compiler's Package for $79.95 a savings of $5.

The Professional version has features designed for the professional or serious user who is going to use the program a lot.
