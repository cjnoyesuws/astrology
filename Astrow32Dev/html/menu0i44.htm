Rebuild File List, (Utilities Menu).

With this version of ASTRO, you can archive files from the list off to another floppy, or disk. To do this copy the file onto another disk and kill the file from the View-Print menu. When you need the file again, copy the file back to your User directory, and select this menu command. This will only work for files generated by ASTRO Version 2.25 or greater since ASTRO as of this version stores file information in each file, and the previous version did not.
