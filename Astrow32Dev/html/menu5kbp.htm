Replace Location.

This Function will Replace the contents of the Record corresponding to this one in the Location Data Base with the contents of the Birth Data Screen, Birth Info Screen, Compatibility Info Screen, Transits Info Screen or Relocation Data screen.

NOTE: this record must have been read with the Get Location menu command, before making changes to it. If you make a change to the spelling, the change of spelling won't be reflected in this list till you run the "Make Popups" menu command located in the Utilities menu.
