ASTRO Report File Viewer.

Use this to view the Text report files that ASTRO produces. If you wish to you can customize the colors and fonts that are used in the viewer.

You can convert this report to text, Print it, Create a Comment File or Create a Cover page from the FILE Menu.

In the Professional Version, you can search for text in the report., and edit text in the report. You mark a paragraph by clicking on the paragraph, click again to edit it or unmark the paragraph. Just double click any paragraph to edit.

If you modify a report, you will be asked it you want to save the report, Pressing Yes Saves, No closes the screen without saving.

See the EDIT Menu.

KEYS:

PAGEUPGo Up One Screen.

PAGE DOWNGo Down One Screen

DOWNGo Down One Line.

UPGo Up One Line.

HOMEGo to the Beginning of Report.

ENDGo to the End of Report.

LEFTMove Cursor Left.

RIGHTMove Cursor Right.

INS or ENTERInsert a New Paragraph.
