Numeric Options, (Options Menu).

This menu command allows you to customize options used when compiling Birth charts, and Producing a Numeric chart. The options set on this screen are directly used when compiling the default Numeric chart. These allow you to use certain features in ASTRO which some users may not wish to use, but you might want, or to turn of other options which you do not feel are valid.This screen also has several options such as "Text" which are not valid in this context since the Numeric chart Produces No Text.
