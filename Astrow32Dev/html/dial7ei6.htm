Location.

This field allows you to select Locations from the above selected country and State in ASTRO's location Database. When you select a location, the information on that Location is read into ASTRO and put into the screen. You can select a choice by either using the Mouse, or by entering the first letter and using the mouse, by typing in the country name into the field, or Pressing Page-Up and Page Down to move to the prev or next Location.
