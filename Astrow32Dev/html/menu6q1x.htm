Save (Birth Data), (File Menu).

This menu command allows you to name a file to save all the Data entered in these menus to. This allows you to save the data you have entered for a future time. If you don't enter a disk or directory, it will be saved to your User disk and Directory. If you save to a file ASTRODAT.DEF, this file will be read Every time ASTRO starts up. ASTRO saves the contents of the screens to the File ASTRODAT.HIS when you quit, and reloads it when you startup again. This is overridden by ASTRODAT.DEF which takes priority if it exists.
