View, (File Menu).

Use this menu command to view an ASTRO File that was created by compiling a horoscope. If you have selected a graphics file, you will view a graphics file, otherwise you will view a text file. This menu command allows you to view the by scrolling through the file like you would with a word processor. You can also do this by Pressing ENTER on the choice you want to view, or double clicking the choice with the mouse.
